# 高圧受電設備 日常点検記録システム

スマートフォン対応の電気設備日常点検記録管理システムです。  
対応設備：高圧受電設備・蓄電池・発電機

---

## ⚠️ 重要：正しい開き方について

このアプリは **HTMLファイルを直接ダブルクリックして開いても動作しません。**

| 開き方 | データ保存 | 動作 |
|--------|-----------|------|
| ファイルをダブルクリック（`file://`） | ❌ 保存されない | 使用不可 |
| メール添付を開く | ❌ 保存されない | 使用不可 |
| AirDropで受け取り開く | ❌ 保存されない | 使用不可 |
| **HTTPS の URL でアクセス** | ✅ 正常保存 | **使用可** |

**→ 必ず以下のいずれかの方法でホスティングしてアクセスしてください。**

---

## アクセス方法（ホスティング）

### 方法1：GitHub Pages（推奨・無料）

1. GitHub にログイン → このリポジトリを開く
2. **Settings** → **Pages** → **Branch: main** → **Save**
3. 数分後に以下のURLが発行される：  
   `https://[ユーザー名].github.io/-electrical-maintenance-report/`
4. そのURLをスマートフォンのブラウザで開く
5. ホーム画面に追加すると、アプリのように使用できる

> **誰がアクセスできる？**  
> → リポジトリが **Public** の場合：URLを知っていれば**誰でもアクセス可能**  
> → リポジトリが **Private** の場合：GitHub 有料プラン（Team以上）が必要。無料の場合は GitHub Pages が使えないため、後述の「方法3」を推奨  
> → **社内限定にしたい場合は方法3（LANサーバー）を推奨**

---

### 方法2：Netlify Drop（最も簡単・30秒で公開・無料）

1. PC で [https://app.netlify.com/drop](https://app.netlify.com/drop) を開く
2. `index.html` をブラウザにドラッグ＆ドロップ
3. 自動的に `https://ランダム名.netlify.app` のURLが生成される
4. そのURLをスマートフォンで開く

> **誰がアクセスできる？**  
> → URLを知っていれば誰でもアクセス可能（点検記録は各端末のブラウザに保存されるため、他の人に見られることはない）

---

### 方法3：社内 LAN サーバー（インターネット不要・社内限定）

PC とスマートフォンが **同じ Wi-Fi** に接続されている場合に使用できます。

**Pythonがある場合（macOS/Linux）：**
```bash
cd [index.htmlがあるフォルダ]
python3 -m http.server 8080
```

**Node.js がある場合：**
```bash
npx serve .
```

**Windowsの場合（コマンドプロンプト）：**
```cmd
python -m http.server 8080
```

起動後、スマートフォンのブラウザで以下を開く：  
`http://[PCのIPアドレス]:8080`  
（PCのIPアドレスは `ipconfig`（Windows）または `ifconfig`（Mac）で確認）

> **誰がアクセスできる？**  
> → 同じ Wi-Fi ネットワーク内のデバイスのみアクセス可能。社内LANなら社内限定になる。

## PC → iPhone への設定引き継ぎ手順

PCで設定した点検カテゴリ・項目をiPhoneに引き継ぐには以下の手順を実施します。

### 手順

**PCで（設定エクスポート）：**
1. ブラウザで URL を開く
2. 「設定」タブを開く
3. 「設定をエクスポート」ボタンを押す  
   → `点検設定_YYYY-MM-DD.json` がダウンロードされる

**JSONファイルをiPhoneに転送：**

| 転送方法 | 手順 |
|---------|------|
| **iCloud Drive** | Finder / エクスプローラーで iCloud Drive に保存 → iPhone の「ファイル」アプリからアクセス |
| **AirDrop** | Mac から iPhone へ AirDrop → 「ファイル」アプリに保存 |
| **Google Drive / Dropbox** | PC でアップロード → iPhone のアプリからダウンロード |
| **メール（非推奨）** | JSONファイルを添付して送信 → iPhone Mail でファイルを保存 → 「ファイル」アプリ経由で選択 |

**iPhoneで（設定インポート）：**
1. iPhone Safari でアプリのURLを開く
2. 「設定」タブ → 「設定をインポート」ボタンを押す
3. 「ファイル」アプリから転送した JSON ファイルを選択
4. 「設定のインポートが完了しました」と表示されれば成功

> **注意：** `file://` や制限された環境で開いている場合、インポート時にエラーメッセージが表示されます。必ず HTTPS の URL からアクセスしてください。

## 基本的な使い方

### 新規入力
1. 点検日・点検者名・設備名称・天候を入力
2. 「計測データ」タブで測定値を入力
3. 「外観点検」タブで各項目のステータスを選択（良好 / 要注意 / 要修理 / N/A）
4. 「保存」ボタンを押して記録を保存

### 履歴・出力
- 過去の記録を月別に検索・閲覧
- 「PDF出力・印刷」で点検報告書を印刷/PDF保存
- 「改修推奨資料」で要修理項目のレポートを出力
- 「年間集計報告書」で年次サマリーを出力
- 「JSONエクスポート」でデータバックアップ

### 設定
- カテゴリ・点検項目・計測値フィールドを自由に追加・編集・削除
- 機器種別（高圧受電設備 / 蓄電池 / 発電機）ごとに個別設定
- 「設定をエクスポート」「設定をインポート」で設定の共有・引き継ぎ

---

## データについて

- **データはブラウザの localStorage に保存されます**
- 同じデバイス・同じブラウザでアクセスした場合のみデータが引き継がれます
- **異なるデバイス間のデータ同期はありません**（JSON エクスポート/インポートで手動共有）
- ブラウザのデータを削除するとすべて消えます。定期的に「JSONエクスポート」でバックアップを取ることを推奨します
